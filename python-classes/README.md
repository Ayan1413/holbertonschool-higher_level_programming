# Python - Classes and Objects

This project covers Object-Oriented Programming (OOP) in Python.

## Tasks

- **0-square.py**: Empty class Square
- **1-square.py**: Square with size attribute
- **2-square.py**: Square with size validation
- **3-square.py**: Square with area method
- **4-square.py**: Square with getter and setter
- **5-square.py**: Square with my_print method
- **6-square.py**: Square with position
