# read_file

A simple Python script that reads and prints the content of a text file.

## Usage

\`\`\`bash
./read_file.py <file_path>
\`\`\`

## Description

This script reads a file from the given path and prints its content to standard output.

- If the file exists and has content, it prints the content.
- If the file is empty, it prints an empty line.
- If the file does not exist, it prints `File not found`.

## Requirements

- Python 3
- No external modules required

## Examples

\`\`\`bash
./read_file.py myfile.txt
./read_file.py /path/to/document.html
\`\`\`
